# subgit

## 実装コマンド

* init</br>
  subgitの初期化
* hash-object</br>
  ./exampleディレクトリ内のファイルをハッシュ化する
* cat-file</br>
  ./.subgit/object内のblobオブジェクトのファイルの中身を見る
